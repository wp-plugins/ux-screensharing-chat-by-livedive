This plugin connects your site with LiveDive, the insanely easy way to talk to your site visitors and watch them use your site live.

== Description ==
The LiveDive plugin makes it easy to connect your site with LiveDive, the user feedback and research tool. LiveDive is the insanely easy way to talk to your site visitors and watch them use your site live. It\'s perfect for usability testing, customer/product development, onboarding, sales, and support.

== Installation ==
1. Upload the ux-screensharing-chat folder to the /wp-content/plugins/ directory
2. Activate the UX Screensharing & Chat by LiveDive plugin by going to the \'Plugins\' page in WordPress, locating the UX Screensharing & Chat by LiveDive plugin and clicking \'Activate Plugin\'
3. Configure the plugin by going to the \'Plugins\' page in WordPress, locating the UX Screensharing & Chat by LiveDive plugin and clicking \'Settings\'

== Frequently Asked Questions ==
How do I install the tracking code on my website?
This plugin handles installation of the LiveDive JavaScript code for you. Just install this plugin and add your Site ID on the LiveDive settings page.

Do I need a LiveDive account?
Yes. You can sign up for a free account at https://livedive.co/signup

What if I need help?
Let us know we\'ll do our best to get you sorted. A good place to start is here: https://livedive.co/support
